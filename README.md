# AI Resume Analyzer

AI Resume Analyzer is a React-based web application that analyzes resumes, calculates ATS scores, identifies missing skills, and matches resumes with job descriptions to help job seekers improve their chances of selection.

## Live Demo

 https://vercel.com/vaishnavibajpai12s-projects/ai-interview-preparation-system

## Features

* Resume PDF Upload
* PDF Text Extraction using PDF.js
* ATS Score Calculation
* Job Description Matching
* Missing Skills Detection
* Resume Feedback Generation
* User-Friendly Interface

## Installation

Clone the repository:

git clone <repository-link: https://github.com/Vaishnavibajpai12/AI-Resume-Analyzer>

Install dependencies:

npm install

Run the project:

npm run dev

## Project Highlights

* Built a resume analysis system inspired by Applicant Tracking Systems (ATS).
* Extracted and processed resume content from PDF files.
* Implemented skill-based scoring and job description matching logic.
* Generated actionable feedback for resume improvement.
* Deployed the application using Vercel.
