# AI-Resume-Analyzer
AI Resume Analyzer using React and PDF Parsing

AI Resume Analyzer is a React-based web application that analyzes resumes and provides ATS scores based on predefined skills.

## Features

- Upload Resume PDF
- Extract Text from PDF using PDF.js
- ATS Score Calculation
- Job Description Matching
- Missing Skills Detection
- Resume Feedback Generation

## Technologies Used

- React.js
- JavaScript
- PDF.js
- Bootstrap
- Vite

## Installation

npm install
npm run dev
